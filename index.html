Courses API

Overview

Courses API allows you to query Wellesley course data with ease. Currently it is built using Node.js and Sqlite. Successful requests will return JSON responses.

Dependencies

  • Node.js
  • SQlite
  • Redis

Getting Started

Run npm install from directory to install all dependencies.
Start the app by running: node index.js
Navigate to: http://localhost:8000/

Usage

The API entry point is at http://localhost:8000/courses.
The following parameters can be used for filtering.

      term - The term the course is offered in. For exmaple, fall term of 2009 will be 200009.
      crn - The crn of the course.
      department - The department that offered the course. Use abbreviation. For example, Psychology will be PSYC.
      number - Course number.
      title - Course title. Both exact and partial matches are supported.
      seats - Seats available.
      enrollment - Current enrollment.
      title - Course title. Both exact and partial matches are supported.
      distribution - Distribution that the course satisfies.
      instructor - Instructor who taught the course. Can use first, last, or full name.
      days - Day that the course meets on. Use M for Monday, T for Tuesday, W for Wednesday, R for Thursday, and F for Friday.

Request

The following query will return all the courses offered by the Political Science department in fall 2009 that satisfy the Social and Behavioral Analysis distribution.
http://localhost:8000/courses/?term=200009&distribution=Social and Behavioral Analysis&department=POL
